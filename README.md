## Installation

1.  Open the **Extensions** sidebar in VS Code (`Ctrl+Shift+X`).
2.  Search for `Purple Space Dreams`.
3.  Click **Install**.
4.  Open the **Command Palette** (`Ctrl+Shift+P`), type `Color Theme` and select **Purple Space Dreams** from the list.
